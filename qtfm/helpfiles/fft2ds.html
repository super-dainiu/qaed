Quaternion Function Reference

fft2ds/ifft2ds

Two-Sided Quaternion Fast Fourier transform

Syntax

Y = fft2ds(X)

Description

fft2ds(X) computes a two-sided Quaternion Fast Fourier Transform of the quaternion matrix X using the transform axis pair j and k for the left and right exponentials respectively.

The transform computed by this function is exactly that of Ell (1992). The related function qfft2ds allows user-specified transform axes.

This function is computed by fast Fourier transforms.

The inverse function is ifft2ds and it has the same calling profile. It differs however, in applying a scale factor of 1/N where N is the number of elements in the array X (the product of the number of rows and the number of columns).

See Also

QTFM functions: qfft2ds, qdft2ds
