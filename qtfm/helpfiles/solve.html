Quaternion Function Reference

solve

Solve symbolic quaternion equation(s)
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

S = solve(eqn, var)

Description

solve(eqn, var) solves the symbolic quaternion equation eqn for the quaternion variable var.

To see how to construct eqn refer to the documentation for the overloaded Matlab® function. (The quaternion function operates by solving the system of four equations in real or complex variables which corresponds to the quaternion equation.)

At present the function cannot handle a system of quaternion equations as the overloaded Matlab® function does. This may be added later.

Examples

Here we see solve finding the solution to a monomial with one term. The solution can easily be seen to be x = a^-1 * c * b^-1, but solve finds this by working with the equation in component form, as a set of four equations. 
>> syms aw ax ay az bw bx by bz cw cx cy cz xw xx xy xz
>> a = quaternion(aw, ax, ay, az);
>> b = quaternion(bw, bx, by, bz);
>> c = quaternion(cw, cx, cy, cz);
>> x = quaternion(xw, xx, xy, xz);
>> eqn = a * x * b == c
 
eqn = - bw*(ax*xx - aw*xw + ay*xy + az*xz)
      - bx*(aw*xx + ax*xw + ay*xz - az*xy)
      - by*(aw*xy + ay*xw - ax*xz + az*xx)
      - bz*(aw*xz + ax*xy - ay*xx + az*xw) == cw & 
        bw*(aw*xx + ax*xw + ay*xz - az*xy)
      - bx*(ax*xx - aw*xw + ay*xy + az*xz)
      - by*(aw*xz + ax*xy - ay*xx + az*xw)
      + bz*(aw*xy + ay*xw - ax*xz + az*xx) == cx &
        bw*(aw*xy + ay*xw - ax*xz + az*xx)
      + bx*(aw*xz + ax*xy - ay*xx + az*xw)
      - by*(ax*xx - aw*xw + ay*xy + az*xz)
      - bz*(aw*xx + ax*xw + ay*xz - az*xy) == cy &
        bw*(aw*xz + ax*xy - ay*xx + az*xw)
      - bx*(aw*xy + ay*xw - ax*xz + az*xx)
      + by*(aw*xx + ax*xw + ay*xz - az*xy)
      - bz*(ax*xx - aw*xw + ay*xy + az*xz) == cz
 
>> r = solve(eqn, x)

r =  ((aw*bw*cw + aw*bx*cx + ax*bw*cx - ax*bx*cw + aw*by*cy .... (LONG)

>> r - a^-1 * c * b^-1

ans = ((aw*bw*cw + aw*bx*cx + ax*bw*cx - ax*bx*cw + aw*by*cy .... (LONG)

>> simplify(ans)

ans =  0 + 0 * I + 0 * J + 0 * K
The final result shows that r is indeed equal to the solution stated above because the difference between the two, after simplification, is zero.
