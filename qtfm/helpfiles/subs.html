Quaternion Function Reference

subs

Substitute terms within symbolic quaternion equation(s)
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

S = subs(expr, old, new)

Description

S = subs(expr, old, new) replaces all instances of the quaternion variable old with the quaternion variable new in expr, which must be a symbolic expression created from a quaternion expression (it's class will be 'sym').
