Quaternion Function Reference

qdft2ds/iqdft2ds

Two-Sided Quaternion Discrete Fourier transform

Syntax

Y = qdft2ds(X, A)

Description

qdft2ds(X, A) computes a two-sided Quaternion Discrete Fourier Transform of the quaternion matrix X using transform axis pair A (direction in 3-space).

The transform axis pair, A must be a cell array of two elements which are pure quaternions (real or complex). They need not have unit modulus (although the transform itself is computed using a unit-modulus axis, so that the axis is a root of -1).

The transform computed by this function is a slight generalization of that of Ell (1992)., which had fixed transform axes of j and k for the left and right exponentials. The related function fft2ds provides the default axes of Ell's original transform.

This function is computed by a rather slow direct evaluation of the QDFT. The related function qfft2ds computes the same result much faster.

The inverse function is iqdft2ds and it has the same calling profile. It differs however, in applying a scale factor of 1/N where N is the number of elements in the array X (the product of the number of rows and the number of columns).
